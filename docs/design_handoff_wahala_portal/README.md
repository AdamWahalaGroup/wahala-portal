# Maintained interface references

The maintained Markdown in this directory is limited to durable interface
guidance that still matches the application:

- [`design-system.md`](design-system.md) — brand tokens and core components.
- [`patterns.md`](patterns.md) — behavior, visibility, accessibility, and
  server-authoritative interaction rules.

HTML design canvases and demo scripts may remain as historical visual assets;
they are not requirements. The former screen-by-screen Markdown handoff and dated design deltas were removed after
implementation because they described retired Leads/Triage workflows and
competed with the current operating model. Git history remains the archive.

Product behavior is governed by [`../OPERATING-MODEL.md`](../OPERATING-MODEL.md),
[`../SALES-PROCESS.md`](../SALES-PROCESS.md), and
[`../ROADMAP.md`](../ROADMAP.md). When a visual reference conflicts with those
documents or the server-side security rules, the visual reference loses.
